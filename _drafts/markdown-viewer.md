---
title: 📄 Markdown Viewer
description: A fast, simple, native markdown document viewer
---
Markdown Viewer does one thing, and one thing well.

Can you guess?

It shows you Markdown files.

It's not an editor and it's not a note taking app.

Open a .md file. Drag one in. Or make Markdown Viewer your default markdown file opener and double click a markdown file in finder.

Up it pops.

There are a couple of features:

A sidebar which shows you other markdown files in the same directory as the file you're viewing, as well as any folders which contain markdown files. The root directory is the markdown file you originally open.

An edit button to open the file in your text/code editor of choice.

A file path bar (which you can access with cmd+L). Paste or type in a path: either a markdown file or a folder with markdown files inside it.

That's it. That's the app.
